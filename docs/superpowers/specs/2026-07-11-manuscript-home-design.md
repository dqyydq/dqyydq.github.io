# 手稿册首页与翻页动效设计

## 目标

将当前平铺的首页首屏改为“安静的学术手稿册”。它服务于 AI 学习笔记与项目记录，不采用中世纪奇幻、游戏道具或科技发光视觉。

## 首屏

- 首次访问时显示合上的浅羊皮纸册，封面只包含“邓权耀”和“学习笔记与项目记录”。
- 页面载入后，封面在约 500 毫秒内淡出并向左轻微滑离；左右内页以 80 至 120 毫秒错峰淡入和上浮。取消整张页面的 3D 翻转与横向压缩，避免折叠面板式的生硬观感。已访问用户不重复播放，直接显示展开状态。
- 左页放置姓名、中文自我介绍和“最近在做”。右页展示最新 Markdown 文章的标题、日期、标签、代码摘录和两条暗红色批注。
- 右页内容从最新文章的 frontmatter 读取；代码和批注为稳定的版式内容，不依赖解析文章正文，避免长文格式影响首页。

## 视觉

- 整体背景为深一点的桌面纸面；书页为干净的暖白羊皮色，使用轻微纸张纹理和内侧书脊阴影。
- 标题使用衬线中文字体，正文和导航使用清晰无衬线字体，代码与日期使用等宽字体。
- 批注使用低饱和暗红，代码块使用蓝灰色便签。无破损边缘、魔法图案、羽毛笔或仿古英文。

## 交互与动效

- 首页展开仅使用 opacity、translate 和阴影变化，不阻塞正文内容，也不依赖 3D transform。
- 导航与文章链接在点击后显示短暂纸页离场动效；浏览器正常导航仍为可靠后备路径。
- 鼠标悬停在右页时仅产生轻微倾斜和阴影变化。
- 小屏幕改为上下两张纸页，不使用 3D 翻转；保证正文与按钮可读可点。
- `prefers-reduced-motion: reduce` 下禁用封面、展开、悬停和离场动画，直接显示页面。

## 验证

- `npm run build` 通过，并继续生成首页、博客、项目和文章路由。
- 检查桌面宽度下书脊、双页和文字不重叠；检查移动宽度下纸页单列显示。
- 检查首次载入、已访问状态和减少动效偏好均能展示可访问内容。
- 改造集中在首页、共享布局和全局样式，博客正文不改变。
